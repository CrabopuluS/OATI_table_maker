Принцип формирования столбцов в отчётной таблице:

	Столбцы для итогового отчёта:
	1. Округ - столбец с строчными заголовками по которому фильтруются строки в таблице, фактически, в этом может столбце может содержаться только 13 строчек по административным округам Москвы с учетом возможных данных у которых не отмечен округ)
	2. Всего __ (Вместо __ выбранный объект контроля по полю "Тип объекта") - в данном столбце в разбивке по округам с указанием общей суммы, должно содержаться количество выбранных объектов контроля. Указанное число сортируется по выбранному объекту из поля "Тип объекта" или по всем объектам сразу и указанные данные берутся из таблицы Общий перечень объектов контроля.xlsx.
	3. Проверено объектов с ДД.ММ.ГГГГ по ДД.ММ.ГГГГ (даты в названии берутся из выбранного отчетного периода) - числа в этот столбец следует брать ориентируясь на количество уникальных значений в столбце "Наименование объекта"
	4. % проверенных объектов от общего количества существующих объектов
	5. % объектов с выявленными нарушениями 
	6. Всего выявленных нарушений - тут необходимо посчитать количество уникальных значений по полю "Идентификатор" в рамках выбранной сортировки по отчетному периоду и сложить их с уникальными идентификаторами в рамках предыдущего отчетного периода в статусах "На устранении" и "На контроле инспектора ОАТИ".
	7. Нарушения, выявленные за отчётный период - это количество уникальных значений по полю "Идентификатор" в рамках выбранной сортировки по отчетному периоду.
	8. Нарушения, находящиеся на контроле, с предыдущего периода - это уникальные идентификаторы в рамках предыдущего отчетного периода в статусах (поле - Статус нарушения) "На устранении" и "На контроле инспектора ОАТИ".
	9. Устранено нарушений - это количество уникальных значений по полю "Идентификатор" в рамках выбранной сортировки по отчетному периоду, отсортированные по полю "Статус нарушения" со значением Снят с контроля
	10. Нарушения на контроле - это уникальные идентификаторы в рамках выбранного отчетного периода в статусах (поле - Статус нарушения) "На устранении" и "На контроле инспектора ОАТИ".
